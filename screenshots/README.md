# Project Screenshots

This folder contains screenshots showing the step-by-step implementation of the AWS VPC + EC2 Networking Project.

## Screenshots Included

1. **vpc-created.png**
   - Shows the Virtual Private Cloud (VPC) created in AWS.

2. **subnet-created.png**
   - Displays the subnet created inside the VPC.

3. **internet-gateway.png**
   - Shows the Internet Gateway attached to the VPC.

4. **route-table.png**
   - Displays the route table configuration allowing internet access.

5. **ec2-running.png**
   - Shows the EC2 instance running inside the subnet.

6. **ssh-connection.png**
   - Shows successful SSH connection to the EC2 instance from the terminal.

These screenshots help demonstrate the practical implementation of AWS networking and EC2 deployment.
